To unlock the MSR 0xE2 register, the CFGLock.uefi file must be moved to the Tools folder. Then you need to copy everything from the CGFLock.plist file to your config.plist file. Reboot. In the OpenCore menu, select CFGLock. 

See details here: https://www.insanelymac.com/forum/topic/344035-cfglock-unlock-msr-0xe2/
